DATABASE SCHEMA REQUIREMENTS:
================================================================================

The seeder expects these tables:

1. campaigns
   - id (serial primary key)
   - name, goal, target_segment (text)
   - budget_usd (integer)
   - start_date, end_date (date)
   - status (varchar)
   - channels (jsonb)

2. campaign_briefs
   - id (serial primary key)
   - campaign_id (foreign key)
   - content (jsonb) - {recommended_channels, primary_message, kpis, rationale}
   - quality_score (float)
   - created_by_agent (varchar)
   - approved_at (timestamp)

3. creatives
   - id (serial primary key)
   - campaign_id (foreign key)
   - channel (varchar)
   - variant_index (integer)
   - headline, body (text)
   - cta, tone (varchar)
   - quality_score (float)
   - safety_passed (boolean)

4. performance_events
   - id (serial primary key)
   - campaign_id (foreign key)
   - channel (varchar)
   - event_date (date)
   - impressions, clicks, conversions (integer)
   - ctr, conversion_rate (float)
   - spend_usd, revenue_usd (float)

5. optimization_recommendations
   - id (serial primary key)
   - campaign_id (foreign key)
   - recommendation_type (varchar)
   - description (text)
   - priority (varchar)
   - expected_impact (float)
